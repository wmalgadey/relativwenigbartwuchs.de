**Nerd Fonts** patches developer targeted fonts with a high number of glyphs (icons). Specifically to add a high number of extra glyphs from popular вАШiconic fontsвАЩ such as [Font Awesome](https://github.com/FortAwesome/Font-Awesome), [Devicons](http://vorillaz.github.io/devicons/), [Octicons](https://github.com/github/octicons), and others.

## Font Patcher Script

### Create your own customized patched fonts

Use the provided [FontForge Python Script](https://github.com/ryanoasis/nerd-fonts#font-patcher) to patch your own font or to generate over `**1.4 million**` unique combinations/variations [(more details)](https://github.com/ryanoasis/nerd-fonts#combinations).

You can even specify a custom symbol font with the `**--custom**` option to include even more glyphs.
