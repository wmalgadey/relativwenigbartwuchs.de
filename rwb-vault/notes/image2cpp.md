**image2cpp** is a simple tool to change images into byte arrays (or arrays back into an images) for use with (monochrome) displays such as OLEDs on your Arduino or Raspberry Pi.

It was originally made to work with the Adafruit OLED library (for which your can find an example sketch for Arduino [here](https://github.com/javl/image2cpp/blob/master/oled_example/oled_example.ino)) but has been expanded by the community to be useful in all kind of (embedded) projects.

More info (and credits) can be found in the [Github repository](https://github.com/javl/image2cpp). This is also where you can report any [issues](https://github.com/javl/image2cpp/issues) you might come across.
